## Getting Started

### Build Model (using freezed)

```

### Build Rest apis (using chopper)
```
flutter pub run build_runner build
```

### Build app icon
```
fvm flutter pub run flutter_launcher_icons:main -f flutter_launcher_icons.yaml
```

## Build splash screen

```
